# Pet Finder #

O Pet Finder é uma aplicação para pets abandonados encontrarem um novo lar.

**Stack**:
* React
* Flask
* MongoDB

**Requisitos** para rodar a aplicação:
* Python
* Flask
* MongoDB
* Node.js

Para rodar a **interface de usuário**: 

```cd frontend```
```npm install```
```npm run```

Para rodar a **API**:

```cd backend```
```set flask_app=app.py```
```flask run```
